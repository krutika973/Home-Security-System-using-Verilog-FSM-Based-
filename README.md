 🏠 Home Security System using Verilog

 📌 Overview
This project implements a Home Security System using Verilog HDL. It uses a Finite State Machine (FSM) to monitor motion, door, and window sensors. The system can be armed or disarmed using a keypad-based password.

 🎯 Objectives
- Design a digital home security system  
- Implement FSM-based control logic  
- Detect intrusions using multiple sensors  
- Provide password-based access control  

 🧠 Working Principle
The system operates in three states:
- DISARMED → System inactive  
- ARMED → Monitoring sensors  
- ALARM_TRIGGERED → Alarm activated  

- Correct password → Arms/Disarms system  
- Sensor trigger → Activates alarm when armed  

⚙️ Methodology
- FSM-based design using Verilog  
- Clock-driven state transitions  
- Sensor-based intrusion detection  
- Testbench used for simulation and verification  

 🛠️ Components (Simulation)
- Verilog HDL  
- Testbench  
- Simulation tool (ModelSim / EDA Playground)

🔄 Features
- Multi-sensor monitoring (motion, door, window)  
- Password-based security 🔑  
- Real-time alarm system 🚨  
- Reliable state transitions  

 🌍 Applications
- Home security systems  
- Office security  
- Embedded alarm systems  
- Smart home solutions  

✅ Advantages
- Simple and efficient design  
- Secure access control  
- Easy to implement  
- Scalable system  

⚠️ Limitations

 📌 Conclusion
This project demonstrates an efficient FSM-based home security system using Verilog. It ensures safety by detecting intrusions and controlling access through password authentication.
